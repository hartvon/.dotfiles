# hartvon's dotfiles 

## Prerequisite

- git
- rsync

## Installation

1. Clone the repo:

```bash
$ git clone git@github.com:hartvon/.dotfiles.git
```

2. Source the `bootstrap.sh` (*may ovewrite your default files, use at your own risk*):
```bash
$ source bootstrap.sh
```
